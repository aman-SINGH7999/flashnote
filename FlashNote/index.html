Flash Notification Documentation

1️⃣ Include CSS and JS

To use the flash notifications, add the following lines inside the <head> tag of your HTML:

<link rel="stylesheet" href="notify.css">
 <script src="notify.js"></script>

2️⃣ Usage

Use the flash object to show different types of notifications:

flash.error(message, position, duration, closeButton);
 flash.success(message, position, duration, closeButton);
 flash.info(message, position, duration, closeButton);
 flash.warning(message, position, duration, closeButton);

3️⃣ Parameters

  • message (required): Text you want to display.
  • position (optional): 'top-left', 'top-center', 'top-right', 'bottom-left', 'bottom-center', 'bottom-right' (default: 'top-right').
  • duration (optional): Time in milliseconds before auto-close (default: 3000ms).
  • closeButton (optional): true or false (default: false) to show/hide close button.

4️⃣ Example Usage

Top Left Error:

flash.error('Something went wrong!', 'top-left');

Top Center Success with Close Button:

flash.success('Operation Successful!', 'top-center', 5000, true);

Bottom Right Info with Custom Duration:

flash.info('This is an info message', 'bottom-right', 1000);
